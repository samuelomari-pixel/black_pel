# black_pel# Blackpel Website - Home Page

## Project Overview
This project is the **Home Page** of the Blackpel website, a technology-based company providing digital solutions to individuals, schools, and businesses.  
The Home Page serves as the **landing page**, introducing the company, its services, and providing navigation to other sections of the website.

---

## Features
- Clean and modern layout with a background image
- Navigation menu linking to Home, About, Services, and Contact sections
- Sectioned content for better readability
- Call-to-action **Sign Up button**
- Responsive and readable text with contrasting colors
- Styled using **HTML and CSS** without any frameworks
- Ready for integration with Login and Dashboard pages

---

## Sections
1. **Home** – Welcome message and company mission  
2. **About** – Overview of the company and its goals  
3. **Services** – List of services including Website Development, System Design, Data Management, and Technical Support  
4. **Contact** – Email and phone contact information  

---

## Technologies Used
- **HTML5** – Page structure and content  
- **CSS3** – Styling, background image, colors, and layout  
- **Flexbox/Grid** – Can be applied for responsive layouts if extended  
- No CSS frameworks used
